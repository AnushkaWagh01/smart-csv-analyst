## Problem
Manual analysis of CSV files is **time-consuming**, repetitive, and often requires technical skills like Excel, Python, or SQL. Many students, analysts, researchers, and small business owners struggle with it.

## Solution
**InsightCSV** is a modern, beautiful dark-themed web app that lets you upload any CSV file and get **instant AI-powered analysis** with clear insights, charts, and actionable recommendations.

## ✨ Key Features
- ⚡ One-click CSV upload with instant analysis
- 📋 5 Structured Sections: Insights, How to Find, Code, Charts & Recommendations
- 📊 Animated Quality Gauge
- 💬 Follow-up Chat (ask questions in natural language)
- 🕒 Analysis History (saved using localStorage)
- 🎨 Modern Dark UI with blue/cyan accents

## 🛠️ Tech Stack
- **Frontend**: Lovable.dev
- **AI Models**: Groq (fast inference) + Gemini (deep analysis)
- **Storage**: localStorage

---

## How It Works
1. Upload any CSV file
2. AI automatically analyzes the data
3. Get well-structured insights + visualizations instantly
4. Continue asking follow-up questions through chat
